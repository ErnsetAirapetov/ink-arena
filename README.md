# InkArena

PvP-игра, где заклинания **рисуют**: чем точнее глиф — тем сильнее заклинание,
а разные глифы складываются в комбо.

Сейчас это песочница — одиночная веб-страница для проверки идеи: рисуешь глифы
(можно несколькими линиями), жмёшь пробел, видишь точность, получаешь эффект и
комбинируешь стихии.

## Запуск

```bash
npm install
npm run dev      # дев-сервер (обычно http://localhost:5173)
```

## Тесты и сборка

```bash
npm test         # юнит-тесты (Vitest)
npm run build    # продакшн-сборка в dist/
```

## Как играть

Зажми мышь/палец и рисуй глифы (можно несколькими линиями):
△ огонь · ~ вода · ◠ воздух · □ земля · ⚡ молния · ○ щит.
Нажми **ПРОБЕЛ** — заклинание распознается и сработает.
Нарисуй два глифа рядом и нажми пробел для комбо: огонь+воздух = огненный вихрь,
вода+молния = шторм, земля+огонь = магма, вода+щит = лечащий барьер.
