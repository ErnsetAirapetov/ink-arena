# Заклинания InkArena

Шпаргалка по всем доступным глифам и комбо на текущий момент. Источник правды —
`src/recognition/glyphs.ts` (глифы) и `src/spells/combo.ts` (комбо).

**Как кастовать:** нарисуй глиф (можно несколькими линиями), затем нажми
**ПРОБЕЛ**. Для комбо нарисуй два глифа рядом и нажми пробел. Сила = насколько
точно совпало с эталоном; у комбо — средняя точность двух глифов.

## Базовые глифы (6 стихий)

| Стихия | id | Форма | Как рисовать |
|--------|----|-------|--------------|
| 🔥 Огонь | `fire` | △ треугольник | вершиной вверх, замкни в исходную точку |
| 💧 Вода | `water` | ~ волна | синусоида слева направо (две «горки») |
| 🌪 Воздух | `air` | ⌇ вертикальная волна | волнистая линия сверху вниз |
| ⛰ Земля | `earth` | □ квадрат | замкнутый квадрат |
| ⚡ Молния | `lightning` | зигзаг | острый зигзаг сверху вниз |
| 🛡 Щит | `shield` | ○ круг | замкнутая окружность |

## Комбо (два глифа рядом, порядок не важен)

| Комбо | id | Из чего |
|-------|----|---------|
| 🌋 Огненный вихрь | `firestorm` | огонь + воздух |
| ⛈ Шторм | `storm` | вода + молния |
| 🌋 Магма | `magma` | земля + огонь |
| ✨ Лечащий барьер | `healing-barrier` | вода + щит |

## Что покажет HUD

- **Одиночное:** `Заклинание: <Имя> · сила N%`
- **Комбо:** `КОМБО: <Имя> · сила N%`
- **Осечка** (повод и текст):
  - ничего не нарисовано — `Ничего не нарисовано`;
  - слишком неточно (`score` ниже `minScore`) — `Слишком неточно — рисуй чётче`;
  - два глифа, которых нет в таблице комбо — `Эти глифы не сочетаются`;
  - больше двух глифов — `Пока не больше двух глифов`.

## Полезно знать при тесте

- Соседние глифы для комбо рисуй **с заметным промежутком** между ними, иначе
  кластеризация склеит их в один глиф. Порог — `clusterGapPx` в `src/config.ts`.
- Порог распознавания осечки — `minScore` там же.

## Управление

- Рисуй глифы мышью/пальцем, **ПРОБЕЛ** — каст.
- **G** — показать/скрыть подложку-обводку (гост-глифы и легенду комбо).
  По умолчанию включена: обведи гост-контур, чтобы получить почти 100% точности.
