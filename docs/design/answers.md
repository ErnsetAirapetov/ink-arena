# Ответы на вопросы рамки

Сырые ответы на `questions.md`. Пиши как думается, своими словами —
структурирование будет отдельным шагом. Формат свободный, но начинай ответ
с номера вопроса, чтобы потом можно было собрать всё по рамке:

```
## A1
Игрок чувствует себя ...

## C1
Думаю, гибрид, потому что ...
Хотя есть сомнение: ...
```

Можно отвечать не по порядку, возвращаться и дописывать. Если ответ меняется —
не стирай старый, допиши ниже «передумал: ...» (история решений ценна).
Если вопрос кажется неправильным — так и напиши, это тоже результат.
